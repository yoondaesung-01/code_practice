# [Baekjoon] 2739. 구구단

## 1. ❓ 문제 정의
- **목표:** 숫자 N을 입력받아 N단을 출력한다.
- **조건:**
    1. N은 1보다 크거나 같고, 9보다 작거나 같다.
    2. 출력 형식은 `N * 1 = N`, `N * 2 = 2N` ... 순서로 `N * 9`까지 출력해야 한다.

## 2. 💡 풀이 접근
- 입력받은 수 N에 대해 1부터 9까지 곱해야 하므로 **반복문(`for`)**이 필요하다.
- 파이썬의 `range(start, end)`는 `end` 직전까지만 반복하므로, 9까지 곱하려면 `range(1, 10)`을 써야 한다.
- 출력 형식이 지정되어 있으므로(`2 * 1 = 2`), 변수와 문자열을 함께 출력하기 편한 **f-string**을 사용한다.

## 3. 📝 정답 코드 (Python)
```python
N = int(input())

(1) for i in range(1, 10):  
    print(f"{N} * {i} = {N * i}") # 방식 알아두기

(2) for i in range(1, 10):
    print(N, "*", i, "=", N * i)
```
## ✅ 체크 포인트 (중요!)
- f-스트링 (f-string) 사용법:
문자열 앞에 f를 붙이면 중괄호 { } 안에 변수나 계산식을 직접 넣을 수 있다. 
```python
f"{N} * {i} = {N * i}"
{N} : 입력받은 단 (예: 2)
{i} : 곱해지는 수 (1~9)
{N * i} : 곱셈 결과 (예: 2, 4, 6...)
```
콤마(,)를 쓰는 것보다 띄어쓰기나 형식을 맞추기가 훨씬 쉽고 직관적이다.

- range 범위: range(1, 9)가 아니라 range(1, 10)이어야 9까지 출력된다는 점을 주의해야 한다.
